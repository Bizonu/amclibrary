* In the x64 folder should be put the .dll files used by the x64 version of the application, and the installer for the Flash Player Active X. Be sure to update these files if another DirectX SDK is used, or another Flash Player is desired.


* In the x86 folder should be put the .dll files used by the x86 version of the application, and the installer for the Flash Player Active X. Be sure to update these files if another DirectX SDK is used, or another Flash Player is desired.
